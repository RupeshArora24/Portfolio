# Portfolio Website

A simple, responsive portfolio website starter built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main page structure and content
- `styles.css` — responsive styling and layout
- `script.js` — mobile menu toggle and dynamic current year

## Customize

1. Replace `Your Name` with your name.
2. Update the hero text, about section, and project cards.
3. Change the email address in the contact section.
4. Add your own case study links or project pages.

## Run locally

Open `index.html` in your browser, or use a local static server if you want a better development workflow.
